# Delay-Doppler-CRB

This is the open-source MATLAB code to generate the numerical results in the paper "On A Unified Cramér–Rao Bound Framework for Joint Delay-Doppler Estimation with Advanced Multi-Carrier Waveforms" by Zi-Jie Wang, Xudong Wang, and Giuseppe Caire.

Note: These scripts only generate raw numerical data for all figures. Plotting scripts are not included; readers may visualize the results using straightforward and standard MATLAB plotting commands if desired. All rights reserved @ National University of Singapore.

To obtain numerical results for Fig. 2, please run "main_CRB/CRB_SNR_isotropy_main.m";

To obtain numerical results for Fig. 3, please run "main_CRB/CRB_data_main.m";

To obtain numerical results for Figs. 4-5, please run "Pareto_Front/main.m";

Thanks for your attention and interest!
